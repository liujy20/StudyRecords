json格式的字符串。

```js
{
    "id":1,
     "name":"小王"
}
```

原生对象

```js
new Object()

    id:1
}
```

 前端为了解析JSON这个格式的数据，浏览器默认封装了JSON对象。

```js
JSON.stringifty()
JSON.parse()

将json转化为js对象
```

```js
java程序中

{
    "id":10,
```

### 三、属性传透

在我们组件通信中，父组件传递数据给子组件

```js
<Child :name="name" :address="address"></Child>
<Child :obj="obj"></Child>
<Child v-bind="$props"></Child>
<Child v-bind="$attrs"></Child>
```

$props和$attrs两则都在组件中的外部数据

```js
<Child :name="name" :address="address"></Child>

Child:
props:["name"]
$attrs可以得到address属性
```
